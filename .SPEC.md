# 聯盟行銷網站建構藍圖

## 專案概述

建構一個以 3C 科技為起點、可擴展至多領域的聯盟行銷部落格網站。  
採用 Next.js + MDX + SQLite 混合架構，部署於 Vercel。

## 技術棧

| 項目 | 選型 |
|------|------|
| 框架 | Next.js 14+ (App Router, TypeScript) |
| 樣式 | Tailwind CSS + shadcn/ui |
| 內容 | MDX (contentlayer2 / next-mdx-remote) |
| 資料庫 | SQLite (better-sqlite3) + Drizzle ORM |
| 部署 | Vercel (免費方案) |
| SEO | next-sitemap + JSON-LD 結構化資料 |
| 分析 | Vercel Analytics + 自建瀏覽追蹤 |

## 網站架構

```
首頁 (/)
├── 文章列表 (/blog)
│   ├── 文章詳情 (/blog/[slug])
│   └── 分類篩選 (/blog/category/[category])
├── 產品推薦 (/products)
│   ├── 產品詳情 (/products/[slug])
│   └── 產品比較 (/compare?items=a,b,c)
├── 關於我 (/about)
├── 搜尋 (/search)
├── RSS Feed (/feed.xml)
└── Sitemap (/sitemap.xml)
```

## 核心功能模組

### 1. 多分類系統 (彈性擴展)
- categories.ts 集中管理分類定義
- 每個分類有獨立 slug、名稱、描述、icon
- 新增領域只需在 categories.ts 加一筆 + 新增對應文章
- 支援子分類 (如 3C > 耳機 > 藍牙耳機)

### 2. MDX 文章系統
- Frontmatter: title, description, category, tags, keywords, affiliateProducts, publishedAt, updatedAt, featured
- 文章內可嵌入自訂元件: ProductCard, ComparisonTable, ProsConsList
- 自動生成目錄 (Table of Contents)
- 自動生成相關文章推薦

### 3. 聯盟連結管理
- 產品資料集中存放在 content/products/ (YAML 格式)
- 每個產品: 名稱、圖片、價格、聯盟連結(多平台)、評分、規格
- 連結點擊追蹤 (SQLite 記錄)
- 聯盟揭露聲明自動插入每篇文章

### 4. SEO 優化
- SSG 靜態生成 (極快載入)
- JSON-LD 結構化資料 (Article, Product, BreadcrumbList)
- 自動 sitemap.xml + robots.txt
- Open Graph + Twitter Card meta tags
- 自動產生 canonical URL

### 5. 數據追蹤 (SQLite)
- 文章瀏覽數
- 聯盟連結點擊數 & 點擊來源
- 熱門文章排行
- 搜尋關鍵字記錄

### 6. 使用者體驗
- 響應式設計 (Mobile First)
- 深色/淺色模式切換
- 文章搜尋 (全文搜尋)
- 相關文章推薦
- 電子報訂閱

## 資料庫 Schema (SQLite)

```sql
-- 文章瀏覽追蹤
CREATE TABLE page_views (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  slug TEXT NOT NULL,
  type TEXT NOT NULL DEFAULT 'blog',  -- blog | product
  viewed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  referrer TEXT,
  user_agent TEXT
);

-- 聯盟連結點擊追蹤
CREATE TABLE affiliate_clicks (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  product_slug TEXT NOT NULL,
  platform TEXT NOT NULL,          -- momo | shopee | amazon
  source_slug TEXT,                -- 從哪篇文章點的
  clicked_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  referrer TEXT
);

-- 搜尋記錄
CREATE TABLE search_logs (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  query TEXT NOT NULL,
  results_count INTEGER DEFAULT 0,
  searched_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);

-- 電子報訂閱
CREATE TABLE newsletter_subscribers (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  email TEXT UNIQUE NOT NULL,
  subscribed_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
  status TEXT DEFAULT 'active'     -- active | unsubscribed
);
```
